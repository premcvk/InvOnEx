# Excel Invoicing Tool

I Prem Chowdhary VK am a young accounting professional made InvOnEx, this excel file a macro enabled xlsm file, Initially to help a few clients of mine to generate, store and retrieve their invoices
InvOnEx is a simple Excel-based invoicing system for freelancers and small businesses.

## Features
- Tax calculation
- Printable invoice format
- Customer database
- Invoice database


## How to Use
1. Open the file
2. Enable macros (if prompted)
3. Enter customer details
4. Generate invoice

## Requirements
- Microsoft Excel 2021 or later

## Limits and Workaround
This utility can be used to generate Invoice for one Charging item (particulars) alone and one rate of tax
Workaround
Incase you want an invoice with few charging items (particulars) or multiple tax rates, you have to raise more than one invoice
